## React

Install mailgo through

```
npm install mailgo
```

or

```
yarn add mailgo
```

and then just

```
import 'mailgo';
```

in `App.js`, `index.js` or any other root component.

## Angular

Install mailgo through

```
npm install mailgo
```

or

```
yarn add mailgo
```

then add mailgo to your `.angular-cli.json` in this way

```
"styles": [
    ...
],
"scripts": [
    ...
    "../node_modules/mailgo/dist/mailgo.min.js"
]
```
